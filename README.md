## Projeto em Destaque

### AgendaPRO

Sistema **multi-tenant** de agendamento para barbearias, clínicas e escritórios — agenda, pagamentos e lembretes em uma única plataforma.

| | |
| :-- | :-- |
| **Mobile** | Flutter · Dart |
| **Estado** | Riverpod · go_router |
| **Backend** | Supabase *(Postgres + Row-Level Security)* |
| **Pagamentos** | Stripe · Pagar.me · Iugu |
| **Notificações** | Resend · Z-API *(WhatsApp)* |
| **Status** | Em desenvolvimento ativo |
